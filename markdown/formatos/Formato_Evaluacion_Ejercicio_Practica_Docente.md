### Objetivos
Valorar el desempeño personal y profesional, e identificar fortalezas y debilidades para establecer estrategias de mejoramiento.

### Instrucciones
Expresar su evaluación de forma cuantitativa (0 a 10) para cada aspecto.

---

| Aspectos | Evidencias | Valor asignado (0-10) |
| :--- | :--- | :--- |
| **Dominio disciplinar** | Domina conocimiento, fluidez verbal, seguridad, correspondencia con plan de estudio, aportes innovadores | |
| **Planeación y organización académica** | Planea clases con elementos pedagógicos/didácticos, propone estrategias/mediaciones creativas, cumple con entregas | |
| **Pedagogía y didáctica** | Fundamentación pedagógica/didáctica, propone estrategias ajustadas a necesidades, programa estrategias diferenciadas para NEE | |
| **Uso de recursos** | Maximiza uso de recursos tecnológicos, diseña estrategias innovadoras con tecnología, habilidad en selección/diseño de recursos | |
| **Evaluación del aprendizaje** | Conocimiento/habilidad en diseño/aplicación de métodos/técnicas/instrumentos de evaluación, programa evaluaciones puntualmente, reflexiona resultados y establece mejoras | |
| **Comunicación** | Manejo adecuado de competencias comunicativas (oral/escrita), mantiene motivación/disciplina sin maltrato, expresa argumentos claros/respetuosos | |
| **Relaciones interpersonales y trabajo en equipo** | Relaciones cordiales/asertivas, construye relaciones colaborativas, participa en proyectos colectivos | |
| **Negociación y mediación** | Oportuno ante conflictos, facilita acuerdos/soluciones, escucha atenta, comprende causas de conflictos con imparcialidad | |
| **Compromiso institucional** | Buena presentación personal, porta distintivos (carnet-uniforme), asiste puntualmente a clases/actividades, cumple con manual de convivencia | |
| **Socioemocionales** | Fomenta actitudes positivas/autoconfianza/iniciativa/motivación, promueve desarrollo personal/sociocognitivo/valores, exhibe comportamiento ético/responsabilidad social, contribuye al fomento de valores humanos, muestra respeto | |

*Nota: el diligenciamiento sin tachones ni enmendaduras.*
